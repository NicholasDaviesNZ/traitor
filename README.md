# traitor

A tool for extracting seed traits from digital images.

## Installation

```bash
pip install git+https://github.com/TankredO/traitor
```

## Usage

```bash
traitor -h
```

### Extraction

```bash
traitor extract -h
```

### Alignment

```bash
traitor align -h
```

### Measurement

```bash
traitor measure -h
```

### Shape Clustering (experimental)

```bash
traitor shape -h
```
